# 第 099 章：秦岭夜线

他们回到江州时，春节假期已经快结束。

城市湿冷，路边的雪化成灰水。程野刚下车，就看见小区门口还有人举着手机等他。方既白的视频让舆论转向，却没有让世界变得温和。

江州没有雪山。

只有高架桥下的风、便利店门口的热包子、学校门口还没拆完的红灯笼。程野站在这些东西中间，背包肩带压出的酸痛还在，手指上黑石磨出的细口也还在。他忽然有一种错位感：山里的冷没有完全退出来，城市却已经要求他回到能被解释、能被采访、能被判断的生活里。

祁放先回去补报告。

方既白去见顾棠家属授权的律师。

沈听岚把档案副本带回住处，所有电子文件离线保存。她没有回父母家。她还不知道该怎样告诉他们，哥哥的声音确实回来了，又仍然没有回来。

这句话没有任何一种亲属能轻易听懂。

叶知微是晚上十点打来电话的。

她只说：“开离线会议。”

五分钟后，几个人重新出现在加密会议里。每个人都很疲惫，却没人问能不能明天再说。

叶知微共享了一段频谱。

“这是秦岭夜线三年前的一段公开搜救音频。”她说，“原本被当成无线电串台。下午我用山里石子点名那段做结构比对，发现它不是普通串台。”

屏幕上两条波形叠在一起。

一个来自梅里。

一个来自秦岭。

频率不完全相同，环境噪声也不同。可点名间隔、尾音被补全的位置、第三次呼叫后多出低平声的结构，几乎一致。

方既白低声说：“不是同一座山。”

“所以更糟。”叶知微说。

她打开第二份资料。

秦岭夜线，民间夜间穿越事故，公开记录里没有重大伤亡，只有一名队员短暂失联后自行下撤。事故复盘附件后来被删除，缓存里残留一行英文路径。

`NightLine-QIN / call-check / roster-cross`

沈听岚看着 `roster-cross`，想起旧档案索引里的 `JM roster cross`。

程野问：“有录音吗？”

叶知微沉默一下。

“有。但你们先听前十秒。”

她按下播放。

城市夜里，电脑扬声器里传出秦岭的风声。那风比梅里低，带着树林和雨水的湿气。

然后是点名。

第一个声音很模糊。

不像任何他们认识的人。

可被点到的名字清楚得不能再清楚。

“程野。”

会议里没有人说话。

程野坐在屏幕前，脸色一点点失去血色。

三年前的秦岭夜线。

他没有去过。

可无线电点名第一个叫的，是他的名字。
