# 第 091 章：舆论初反转

方既白把素材导出来时，手在发抖。

不是冷。

是克制。

他第一次发现，删素材比拍素材难。

拍的时候，人总能给自己找理由：先留下，回去再判断；先记录，发布时再克制。可真正面对时间线，面对顾棠发紫的嘴唇、沈听岚挡镜头的手、程野被围堵时没有表情的脸，他才知道“回去再判断”不是道德，它只是把决定推迟。

他删掉了顾棠失温时的近景，删掉沈听岚扶着顾棠母亲的画面，也删掉了程野被围堵时那些最容易被剪成争议的视频。

最后留下来的只有几段必要证据。

报备语音。

顾棠求救标记的局部。

祁放交接医疗的背影。

路线记录里被标注为“不可复走段”的缺口。

还有一段他自己对镜头说的话。

“这次行动不是探险，不是挑战，不是攻略，也不是任何人可以复制的路线。我们不会公开坐标和路径。顾棠活着，是因为她自己一直在求生，也因为当地、家属和现场人员都在拉她。不要把这件事剪成一个人的神话。”

他说完后沉默了两秒。

又补了一句。

“也不要把五年前的事，用今天一条视频判完。”

视频发出去后，评论区前半小时全是混乱的。

有人骂他遮掩，有人骂程野洗白，有人质问品牌为什么让轻装队脱离后勤，也有人开始翻出顾棠失踪前负责的安全文案和薛临团队的拍摄预告。

舆论像一块冻住的冰，被从内部敲开第一道裂缝。

程野没有看手机。

祁放强制所有人补水、吃东西、换干衣服。沈听岚坐在旅店走廊尽头，把证据按“可公开”“仅交家属”“仅交当地”“涉及旧案暂封”四类重新编号。

这一次，分类不只是资料习惯。

它也是边界。可公开的东西要能阻止后来者误解；仅交当地的东西要能帮助真实救援；仅交家属的东西要保住顾棠作为人的尊严；涉及旧案暂封的东西，不能因为他们急着证明自己，就被做成下一条可以复走的线。

她把方既白发出的那段视频看完，没有夸他。

只说：“顾棠的脸没有露。”

方既白点头：“我知道。”

“旧雪门也没有。”

“没有。”

沈听岚这才把手机扣下。

夜里十点，江州大学发布声明。

声明写得很稳。

对五年前登山社旧案表示遗憾。

对社会关切表示重视。

称学校多年来持续配合调查，相关旧案资料已经完整移交有关部门，不存在隐瞒、遗漏或拖延。

最后一行说：请公众勿信未经证实信息。

程野看完那句话，没说话。

沈听岚把声明截图放大。

她盯着“完整移交”四个字，眼神一点点冷下来。

“他们在撒谎。”她说。
