# 第 045 章：不能这么撤

沈听岚那句话之后，队伍安静了很久。

程野没有为自己辩解。风从赤岔沟口吹上来，掀动他外套下摆。他站在那里，像一个被迫再次接受审问的人，也像一个不能离开岗位的人。

最后是洛桑开口。

“吵完没有？”

这句话不重，却把所有人从情绪里拽出来。

沈听岚的脸色仍然冷，程野的脸色也不好看。可他们都知道，洛桑不是在让他们和解。他只是提醒：这里不是山下，不是可以把旧账摊开讲清楚的屋子。风不等他们，水声不等他们，那个被反复问起的点名也不等他们。

祁放说：“现在判断。”

程野点头。

他把纸图铺在黑石上，用石头压住四角。风把纸角掀得轻轻响，像它也不愿意待在原位。沈听岚站在一旁，没有帮他压图。她怕自己现在伸手，会像一种原谅。

程野没有看她。

“我们现在有三条选择。”他说，“第一，按手机轨迹原路退。已经试过两次，回到同一飘带，不走。”

方既白轻轻吸了口气。

“第二，按旧图往白石风口退。旧图和现实水线不一致，且经过刚才碎石线，沈听岚已经触发过一次声音诱导。现在队伍情绪不稳，不走。”

沈听岚手指收紧。

他说的是事实，没有把责任推给她。可事实本身就够刺。

“第三，沿赤岔沟边缘找能撤的低处，不进沟心，不追声音，不追影像，遇到新标记只复核不回应。目标不是继续核查白塔，是找回能被洛桑确认的外缘线。”

祁放问：“风险？”

“可能更接近乌蟒线边缘。”程野说。

“收益？”

“不做第三次逆行。”

这个理由很短。

却足够。

洛桑看着纸图：“我带前面。到浅色碎石前停，水声换方向也停。谁也不要抢我的脚印。”

程野说：“你判断能不能走。我判断队伍还能不能走。祁放判断人还能不能走。叶知微判断窗口。任何一个人说停，都停。”

方既白举了一下手。

几个人都看向他。

“我呢？”他问。

程野说：“你判断自己有没有想拍。”

方既白愣了一下。

程野继续：“想拍，就告诉我。别自己扛。”

方既白张了张嘴，最后点头：“行。”

这是入山以来，他得到的最像职责的一句话。不是让他别添乱，不是让他把相机关掉，而是把他的冲动也当作队伍风险的一部分。方既白忽然觉得这比骂他更难受。

沈听岚问：“我呢？”

程野终于看她。

“你判断自己有没有听见沈砚。”他说，“听见，就说。不要再自己处理。”

这句话让沈听岚眼眶发热，但她没有让它显出来。

“你凭什么要求我说？”她问。

程野看着她：“凭我刚才也把接不上的那段说出来了。”

这不是和解。

是交换风险。

沈听岚沉默几秒：“好。”

祁放把医疗包背紧：“那就走第三条。但我先说，一旦人出现明显失温、高反、受伤或者判断力下降，我会强制停。”

“可以。”程野说。

“你也一样。”祁放盯着他，“别把自己排除在外。”

程野笑了一下，很淡：“知道。”

这笑不是轻松，只是他还活着的证据。

队伍再次动起来时，天已经开始偏暗。不是傍晚，而是云层压低后把下午提前折成灰色。赤岔沟口的水声在左前方，白塔前方向被他们抛在身后，蓝白飘带也渐渐看不见了。

走出十几分钟后，沈听岚低声说：“程野。”

程野没有回头，只放慢半步：“说。”

“我刚才听见我哥了。”她说，“现在没有。”

“收到。”

两个字。

没有安慰，没有追问。

却让她稳了一点。

方既白也开口：“我想拍。”

程野问：“拍什么？”

“前面那片反光。”方既白说，“不像水。”

所有人停住。

前方沟口边缘，暗色灌木和碎石之间，有一点很冷的反光。不是溪水那种细碎光，也不是冰面的大片亮，而是规则的、平的、像某种人工材料反射出来的光。

洛桑蹲下看了很久。

祁放压低声音：“帐篷？”

程野没有立刻答。

他看着那点反光，心里有一种很不好的熟悉感。五年前某个傍晚，他似乎也在这种灰光里看见过类似的反光。那时有人说前面有营地，有人说终于能停，有人说先点名。

记忆到这里又断了。

程野吸了一口冷空气。

“能撤我会撤。”他说。

这句话不是说给别人听，像是先把自己钉住。

然后他看向祁放、洛桑和沈听岚。

“现在这条路不能这么撤。”

没人反驳。

风从前方吹过来，带着一点塑料布被冻硬后的轻响。

那点反光在云影里闪了一下。

像一顶失踪队营地里的帐篷，正等在乌蟒线边缘。
