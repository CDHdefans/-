# 洛桑带客与本地山路素材库

当前基准：v0.8 浴火重生版。

第 0 章保留为朦胧记忆锚点。本素材库只服务后续山下和山上段重构，不替第 0 章解释旧案。

## 核心口径

洛桑多吉不是神秘 NPC，也不是规则说明书。他是本地人、向导、带客者、旧搜救参与者、家里有火塘和牲口的人。他的经验来自日复一日看风、水、雪面、脚印、动物状态、客人脸色和山里事故。

他爱山，不是因为把山当成谜题，而是因为家在这里。程野爱山来自同学和队伍，洛桑爱山来自出生、生活、亲人、客人和旧搜救。

## 牧民房/牧屋住宿点与木棚/驮棚短停点

当前核查队成立的关键不是“过夜赌运气”，而是两天一夜低线过夜核查 + 最低过夜系统 + 洛桑确认并获许可的牧民房/季节性牧屋。

- 牧民房/季节性牧屋可以住人，但只是简陋庇护，不是补给点。
- 木棚/驮棚不是安全屋，最多短暂停靠、卸驮、挡一阵风或放草料驮具。
- 可住的房屋里可以有旧柴灰、破桶、木板、干草、石窝和旧火塘。
- 不能凭空出现大量食物、燃气、睡袋或医疗物资。
- 房屋能提供的是挡风、干地、短时烧水和“让人不被风打穿”的机会；真正的保暖来自队伍自带睡袋、防潮垫、热水和晚间补给。
- 专业向导不会把计划说成碰运气。他会在出发前确定计划住宿点、备用住宿点、最晚折返点和回撤条件；到不了就撤或停，不把木棚当住宿方案。

三夜对应：

- 第一夜：低线牧民房，洛桑提前知道位置，并在山下打过招呼。
- 第二夜：失踪队营地外侧备用季节性牧屋/马帮房，不进入五顶帐篷。
- 第三夜：黑石背风坳没有可靠牧屋，队伍真正进入事故边缘。

## 洛桑可讲的带客故事

### 摄影客等日照

起因：有客人为了拍日照金山，天没亮就想往更高处走。

意外：夜里低云压下来，客人觉得“反正看不到，不如换机位”。洛桑按住不让走，只让他们在木屋边清理前一队留下的垃圾。

解决：大家不情愿地收拾垃圾，烧水，等天亮。第二天早上云开，雪尖亮起来。

结果：洛桑后来常说，山给不给你看，不是你追出来的。有时候你留下来把该做的事做完，天自己会开。

### 雨崩两次

起因：一队外地学生第一次来，只想看照片里的雪山；第二次来，已经听过卡瓦格博背后的故事。

意外：第一次遇到大雪和雪崩消息，一周什么也没看见；第二次一周晴天，看到了梅里最好看的部分。

解决：洛桑没有说“你们虔诚所以晴天”，只说这次他们听得进话，知道什么时候停，什么时候不问不该问的路。

结果：这个故事可用来写程野和洛桑对山的爱不同但都真诚。

### 摄影团轻装后勤错位

起因：拍摄团觉得某个野机位“就在旁边”，想让马帮先去营地，自己轻装绕一下。

意外：看似旁边的坡绕过去就是沟，沟里风声会骗距离。后勤和人错开后，轻装立即变成危险。

解决：洛桑曾经强硬否决过一次，客人不高兴，但当天夜里变天，后来专门回来道歉。

结果：可对应薛临队当前案，说明向导不是万能，真正危险在临时改线和侥幸心理。

### 牲口不肯过

起因：有客人急着赶路，觉得天气还行。

意外：骡马在一段暗冰坡前不肯走，马帮也不愿强拉。

解决：洛桑让客人退回背风处等，后来发现坡下有空响和水线。

结果：洛桑判断边界，不只靠人眼，也看动物状态。

### 牧屋钥匙

起因：一处季节性牧屋不是谁都能用，要提前问人、说明为什么去、什么时候回来。

意外：外来队伍曾经没打招呼就住，留下垃圾和火塘灰，引起村里反感。

解决：洛桑带当前队伍使用牧屋时必须交代许可、最晚到达时间、清理垃圾、火塘复原。

结果：低线过夜核查不是钻空子，而是本地关系、住宿许可和回撤纪律共同成立。

## 搜救与不能外说

洛桑参与过五年前外围搜救。他知道程野归来位置不合理，也知道沈砚曾问过不该问的旧线。但他不能把山里的规矩说成公开攻略。

圈内口径：

- 对外说天气、失温、迷路、坠落、通信失败、路线不清。
- 对内说有些地方不能喊，有些名字不能回，有些线不能补成完整路线，有些点名不能应。
- 规矩要靠现场感觉、经验传承和行动纪律理解，不能被媒体、路线包或模型写成可复走样本。

救援伦理：

- 山没错，人也要救。
- 越界、拍摄、破坏、误判都不能成为现场不救的理由。
- 救援是在承认边界后，把还活着的人带离危险。
- 救援失败不等于不努力；有时是天气、无痕迹、规则边缘和救援者安全共同逼停。

## 洛桑说话方式

- 平时热情，爱劝饭、倒茶、讲客人的糗事。
- 到旧线和旧搜救相关话题时变短句。
- 不解释机制，只给动作：“停”“不要喊”“等风转”“日照到石头再走”。
- 紧张时看雪尖、按旧珠串、摸火塘灰、检查门闩。
- 他不说“山惩罚人”，可以说“这个地方不喜欢人把话说满”。

## 正文嵌入建议

- 山下第 13-28 章补洛桑家、茶、饭、鞋、火塘、村里关系和牧屋许可。
- 入山前让洛桑明确计划牧民房、备用季节性牧屋、木棚/驮棚只能短停，解决过夜逻辑。
- 正常路段中让洛桑讲带客故事，不要只在异常点说规矩。
- 第三夜让洛桑也紧张：话变短、反复看雪尖和黑石，不写成淡定大师。
- 顾棠出现后，洛桑不评价她做错了什么，只和祁放一起判断怎么把她移出去。
