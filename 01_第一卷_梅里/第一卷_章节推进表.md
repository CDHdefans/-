# 第一卷章节推进表：百章扩容版

规划：以 100 章阶段表作为组织工具，不设硬性章数和字数上限。叙述完整、细节充分、人物和户外流程成立优先。每 8-10 章形成一个小高潮，每 18-22 章形成阶段反转。当前正文以 `02_章节草稿_新版/第一卷_梅里雪山/` 为唯一版本。

情绪刻度：推进 / 喘息 / 高压 / 骗局 / 悲情 / 恶寒。
