# 章节写作模板

每次请 AI 协助写某章时，复制下方任务卡，填写完整后一起发送。

---

## 任务卡（复制这段）

```
## 章节写作任务

章节：[章节编号，例：8]
标题：[章节标题，例：第一处错位]
功能：[章节功能，从推进表里复制]
核心内容：[从推进表里复制]
结尾钩子：[从推进表里复制]
情绪刻度：[高压/推进/喘息/骗局/悲情/恶寒]

视角人物：[谁的视角？程野/沈听岚/多视角切换]
规则出现：[本章触发或提示了哪条规则，编号参考规则民俗档案]
禁止出现：[本章不能提前揭示的信息，参考伏笔主档和一致性清单]

前情摘要（上一章结尾）：
[用 2-3 句话描述上一章结尾的状态]

本章已有草稿：
[如果有，粘贴在这里；没有就写"无"]

字数目标：[约 XXXX 字]
语气要求：[例：程野视角，节奏偏快，不做心理独白，用动作和对话推进]

转场触发：[上一章到本章为什么换地点/见新人/开会议？例：一通电话、短信、邮件、纸质通知、约见、搜救消息、天气窗口变化]
```

---

## 使用说明

1. **填完整**：空白项越多，输出越容易跑偏。"禁止出现"和"前情摘要"最关键。
2. **规则出现**：写规则时参考 `00_总纲/规则民俗档案.md`，确认用哪个版本。
3. **转场触发**：每章换地点、换人物或换信息源，都要有具体触发，不能像资料夹切换。电话、邮件、纸袋、约见、车程、报备回执、天气窗口都可以承担转场。
4. **人物冲突**：写队伍争执或旧案回忆时参考 `00_总纲/人物关系压力表.md`，确认每个人质疑程野的方式不同。
5. **情绪刻度**：
   - **高压**：紧张，读者不能停，不要写环境，多写感知和动作
   - **推进**：信息递进，中等节奏，可以有解释性对话
   - **喘息**：缓和，人物关系，可以有轻松细节
   - **骗局**：某个认知被颠覆，写法是先正常铺垫，在结尾引爆
   - **悲情**：人物内在代价，节奏要慢，用细节而不是直接说
   - **恶寒**：超自然不适，关键是写得"像现实但不对"
6. **写完后更新**：
   - 在 `写作进度.md` 里把状态改成 `[~]` 或 `[x]`
   - 在 `一致性红线清单.md` 里记录本章引入的关键细节
   - 如果本章揭示或修改了伏笔，更新 `伏笔与谜题主档.md`

---

## 示例任务卡（第 8 章）

```
## 章节写作任务

章节：8
标题：第一处错位
功能：第一处错位
核心内容：队伍经过白塔，地图上没有，照片里却像一直存在。
结尾钩子：沈听岚发现白塔上的布条是哥哥惯用结法。
情绪刻度：骗局

视角人物：程野为主，末尾切沈听岚
规则出现：规则 M-01 已在第 7 章铺垫，本章触发第一层错位（地图与现实不符）
禁止出现：不能提沈砚主动越线；不能让程野意识到已进入规则区（他现在还以为是地图错误）；不能让洛桑多吉解释白塔为什么不在地图上

前情摘要（上一章结尾）：
第 7 章结尾，方既白拍到远处雪线上有三个人影，但当时没人注意到。队伍刚过垭口，洛桑多吉说了"不要说话"的规矩，方既白一边遵守一边觉得这很荒谬。

本章已有草稿：
无

字数目标：约 3000 字
语气要求：程野视角，偏冷静，他把错位优先归因于地图误差。场景要写出白塔在雾气里的实体感，不要虚化。结尾切到沈听岚时，她的发现要用她的动作传达，不用心理独白。
转场触发：第 7 章结尾队伍过垭口，洛桑要求停步；本章从停步后的复核和方既白查看照片进入。
```
