# 《规则海拔线》

当前基准版本：v0.8 浴火重生版。
第一卷《梅里》进入系统性重构期：不是推翻原故事气质，也不是给已写章节打补丁，而是重新锁定物资、时间线、人物羁绊、规则因果和读者视角，让后续正文能稳稳往前走。

## 读者入口

五年前，江州大学登山社十二名学生在寒假进入梅里雪山外围一条不该公开的旧线。风雪之后，只有领队程野从低海拔林线醒来，鞋底沾着不属于雪线的湿泥。他说不清自己怎么回来，也找不到其余队友。学校封存档案，舆论把他钉成“把同学带进死路的人”。

五年后，春节前的梅里外围又有一支拍摄队失联。家属收到半截消息，搜救频段里却传来五年前副队沈砚的声音。沈砚的妹妹沈听岚带着录音找上程野。为了核查旧路口、救回仍可能活着的人，程野不得不和她、救援协作人祁放、影像调查者方既白、气象数据研究者叶知微，以及本地向导洛桑多吉一起重返梅里。

他们最初只是外围线索核查队，却很快发现山里的路线、时间、点名和记忆都开始错位。那些规矩不像凭空出现的怪谈说明，更像山里人反复提醒的经验：有些垭口过了不能立刻说话，有些名字听见了不能回头，有些下撤走到第三次，就不再通向山外。

山不害人，山只是有边界。真正让人陷进去的，是被误用的资料、被流传的旧线、没有闭合的事故，以及试图把敬畏边界测绘成安全路线的人。

## 第 0 章锚点

第 0 章保留为第一卷气质锚点：朦胧记忆、断片、雪山余震、程野无法说清的归来。
后续大修必须承接这种“不清楚”的开场，不把五年前具体发生了什么提前解释给读者。旧案真相是作者层的隐形骨架，不是前半卷的说明书。

## 创作红线

- 第 0 章不改成清晰旧案说明。朦胧、断片和无法解释是它的价值。
- 旧案真相不在第一卷前部泄底。读者先看到当前案、录音、旧物、路线污染和记忆回潮，再逐步拼出更深的东西。
- 阶段和章节是管理工具，正文里不跳出来说“第几阶段要出现什么”。正文只跟着人物行动、判断、犯错、复核和选择推进。
- 规则不能像游戏关卡刷新。每次异常都要从现实动作触发：追声、点名、回放、触碰旧物、破坏标记、补完整记录、误把观测点当路线。
- 当前队伍是两天一夜低线过夜核查，不是完整重装长线，也不是纯当天往返或日包轻装。不带帐篷的前提是洛桑确认并获许可的牧民房/季节性牧屋确实可住；每人必须携带睡袋、防潮垫或泡沫垫、保暖层，队伍携带炉具、燃气、热水、晚间补给和应急食。木棚/驮棚只能短暂停靠、挡风或放草料驮具，不能作为住宿点。
- 旧案队伍是学生重装队，有准备、有社团经验、有羁绊，不写成作死队伍。
- 救援者不是审判者。越界、拍摄、破坏、误判都不能成为现场不救人的理由。
- 白线不是低级反派，也不是无辜旁观者。它有救援信念、项目利益、遮掩动机和旧案责任，但第一卷前期只露资料链、设备、缓存和档案痕迹。

## 重构默认

- 允许大改逻辑明显不成立的章节，尤其是物资、时间、场景衔接、规则因果和人物功能化问题。
- 不做补丁式缝合。需要重排动作链、删掉解释性设定、补正常路段时，优先让故事重新长稳。
- 人物故事必须依山而起：越野跑、冬训、重装线、高原长线、倒霉绰号、兄弟绑定、暧昧和冲突，都是程野找回记忆的一部分。
- 写山要有正常时刻：走路、停歇、补水、看脚、调包、等云散、看卡瓦格博、听洛桑讲带客旧事。异常要长在这些正常时刻里。
