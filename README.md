# 《规则海拔线》项目索引

当前基准版本：v0.6，梅里百章扩容 + 户外悬疑操作系统 + 规矩闭合机制 + 白线善意测绘者版。

一句话卖点：一个背负大学队山难骂名的年轻领队，五年后重返梅里雪山外围，在一条会错位、替身、召回人的高山规则线里，寻找失踪者，也寻找当年那场事故被封存的真相。

## 当前创作默认

- 第一卷写梅里雪山地区气质，但核心事故发生在虚构侧谷、虚构垭口、虚构雪线。
- 主角程野现年 26 岁，五年前 21 岁，是大学登山社队长，不是商业领队。
- 旧案是社团私约越界进山，学校事后兜底并封存资料，程野背负“害死同学”的舆论骂名。
- 程野与沈听岚是强羁绊慢热线，第一卷以互相怀疑、被迫同行、共同救人为主。
- 第一卷按长卷写法推进，不设硬性上限；现有 1-24 章为旧版核心稿，后续拆分迁移，不再沿 48 章旧结构继续写。
- 第一卷季节锁定为 1-2 月寒假春节前后；当前案和五年前旧案都在这个窗口，冬季低温、短日照、暗冰、燃气和春节现实压力必须进入正文。
- 沈听岚专业锁定为地理信息/遥感制图 + 音频资料整理；方既白负责影像发布和舆论判断。
- 沈砚第一卷不可确认实体显形，只能通过声音、遗物、影像噪点、旧队手势和转述出现。
- 规则怪谈必须像山里的经验，不能像凭空写出来的游戏说明。
- 世界线采用前中期中国山系、中后期正式海外卷；第一条海外卷为虚构喜马拉雅圣山“央隆雪山”。
- v0.6 锁定：山不害人，山只是有边界；坏了规矩，边界才会变成危险。白线计划不是传统反派，而是把敬畏边界误写成可复现模型的善意测绘系统。
- “大山发脾气”可作为人物口语，机制层必须写成规矩闭合反应，不写神罚。
- 意图回响采用“坏规矩后的感知偏差”：山不替作者判善恶，但傲慢、猎奇、破坏和征服欲会让人更容易听错、看错、走错。
- 每卷按三套系统并行推进：户外行动系统、悬疑谜题系统、人物情感系统。任何进山行动都要有路线、天气、补给、通信、撤退、医疗和权限。
