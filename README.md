# 《规则海拔线》

当前基准版本：v0.6，新版梅里长卷 + 户外悬疑操作系统 + 规矩闭合机制 + 白线善意测绘者版。

## 故事简介

几年前，江州大学登山社十二名学生在寒假进入梅里雪山外围一条不该公开的旧线。风雪之后，只有领队程野从低海拔林线醒来，鞋底沾着不属于雪线的湿泥。他说不清自己怎么回来，也找不到其余队友。学校封存档案，舆论把他钉成“把同学带进死路的人”。

五年后，春节前的梅里外围又有一支拍摄队失联。家属收到半截消息，搜救频段里却传来五年前副队沈砚的声音。沈砚的妹妹沈听岚带着录音找上程野。为了核查白塔垭旧路口、救回仍可能活着的人，程野不得不和她、救援协作人祁放、影像调查者方既白、气象数据研究者叶知微，以及本地向导洛桑多吉一起重返梅里。

他们最初只是外围线索核查队，却很快发现山里的路线、时间、点名和记忆都开始错位。那些规矩不像凭空出现的怪谈说明，更像山里人反复提醒的经验：有些垭口过了不能立刻说话，有些名字听见了不能回头，有些下撤走到第三次，就不再通向山外。

山不害人，山只是有边界。真正让人陷进去的，是被破坏的规矩、未闭合的旧案，以及试图把敬畏边界测绘成路线的人。

### 现在写到哪里

- 当前正文只认新版目录：`02_章节草稿_新版/第一卷_梅里雪山/`。
- 第一阶段已有新版第 0-12 章。
- 第二阶段第 13-28 章已完成一轮正文。
- 第一卷采用长卷写法，不设硬性章数上限；章节推进表里的 100 章只是组织工具。

### 故事核心

- 第一卷写梅里雪山地区气质，但核心事故发生在虚构侧谷、虚构垭口、虚构雪线，不写真实主峰登顶冒险。
- 主角程野现年 26 岁，五年前 21 岁，是大学登山社队长，不是商业领队。
- 五年前旧案是江大登山社 12 人寒假重装徒步/采风私约越界，计划 5 天 4 夜、按 7 天自给准备；事故后只有程野回来，学校善后并封存资料，程野背负“害死同学”的舆论骂名。
- 当前案是薛临团队春节前后拍摄梅里外围轻重装结合路线，因临时绕向野机位导致轻装队和马帮后勤断开，顾棠最早意识到路线不对并留下半截消息。
- 当前行动队不是正式救援队，而是外围线索核查队；真正进入救援，是入山后发现活人痕迹和下撤路线污染。

### 人物默认

- 程野生活里松弛、有活气、能撑场；进山后严肃、短句、专业判断，核心成长是把敬畏写成路线边界。
- 沈听岚是女主与 CP 核心，专业锁定为地理信息/遥感制图 + 音频资料整理；她不提前信任程野，理解来自共同判断和共同承担。
- 沈砚第一卷不可确认实体显形，只能通过声音、遗物、影像噪点、旧队手势和转述出现；底层动机是保护、提醒、纠偏。
- 祁放是顾棠家属侧救援协作人，负责安全刹车、医疗、报备和撤退否决。
- 叶知微是气象/地形/数据位，私人钩子是“异常不能被清理成正常数据”；她坚持“看见线不等于画出路”。
- 方既白是影像与舆论位，私人负债是曾把类似野线拍成诱惑；他的成长是学会什么时候不拍。
- 洛桑多吉是热情健谈的本地向导，不是神秘 NPC；只有触及乌蟒线、沈砚问路和程野五年前归来位置时才收住。

### 世界观默认

- 山不害人，山只是有边界；坏了规矩，边界才会变成危险。
- “大山发脾气”可以作为人物口语，机制层必须写成规矩闭合反应，不写神罚。
- 规则怪谈必须像山里的经验，不能像游戏说明；每个灵异点都要能追溯到路线、天气、补给、通信、撤退、医疗、记录或人际关系的失效。
- 白线计划不是传统反派，而是“善意测绘者”：他们想测绘边界、整理事故、建立救援模型，却把敬畏边界误写成可复现模型。
- 世界线采用前中期中国山系、中后期正式海外卷；第一条海外卷为虚构喜马拉雅圣山“央隆雪山”。
