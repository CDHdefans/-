# 时间轴

两条时间线并行管理：旧案时间线（五年前）和当前时间线（第一卷进行时）。当前版本为新版梅里长卷。

## 旧案时间线（五年前，程野 21 岁）

具体年份由正式写稿时统一确定，正文暂用“五年前”。旧案锁定发生在 1-2 月，大学寒假与春节前后。

| 顺序 | 事件 | 说明 |
| --- | --- | --- |
| T-10 | 江州大学登山社寒假开线讨论 | 名义路线为寒假转山/高海拔徒步采风，程野发起，沈砚任副队 |
| T-9 | 准备会与路书 | 有原始路书、装备清单、队员分工、撤退点、保险和紧急联系人 |
| T-8 | 公装借用 | 旧案锁定为重装徒步：五天四夜按七天备，炉头、帐篷、对讲机、医疗包、辅绳/路绳、食物燃气和飘带材料从社团公装室借出；校级北斗盒子/卫星信标因私约性质、审批和押金流程未取得 |
| T-7 | 沈砚私下向洛桑多吉问旧线 | 洛桑没有追问，后成为他的私人负债 |
| T-6 | 队伍抵达梅里外围 | 12 人在寒假春节前后私约进山，名义路线仍能被解释为外围支线 |
| T-5 | 临时改线讨论 | 有人知道完整计划，有人只知道备用线，有人现场才意识到路线变化 |
| T-4 | 队伍进入乌蟒线边缘 | 路线与白塔垭、赤岔沟、照骨坪相关 |
| T-3 | 沈砚主动越线 | 他收到不该存在的求救信号，临时把队伍推向更高优先级海拔线 |
| T-2 | 点名、风雪、路线记录断裂 | 无线电、纸图、点名表和影像记录从这里开始互相矛盾 |
| T-1 | 队伍全员失踪或进入召回层 | 公开记录为 12 人失踪，实际存在第 13 次应答 |
| T0 | 程野独自归来 | 他醒来时已在低海拔林线，记忆断裂，无法解释归来路线 |
| T+1 | 外围搜救与学校兜底 | 洛桑参与外围搜救，见过程野不可能的归来位置 |
| T+2 | 学校善后、赔偿、舆情和档案封存 | 邵明川主导，陆清和参与；封存真正路线记录、无线电录音和最后点名表 |
| T+3 | 程野背负骂名 | 休学、退社、行业半封杀，靠装备测评、路线顾问和外围救援协助维生 |
